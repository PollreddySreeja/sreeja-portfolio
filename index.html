  • Hybrid CNN–Random Forest model with 200 estimators
  • Feature vectorization with 3072 features per image
  • 80–20 train-test split with precision/recall/F1 metrics
  • Tkinter GUI for instant clinical prediction

  • CNN with Firefly Optimization for hyperparameter tuning
